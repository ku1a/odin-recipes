## What this is:

This is the first project encountered when doing The Odin Project web development course. This is a recipe browser (so to speak) written in plain html.

Recipes are viewed by clicking on links which takes the user to the appropriate recipe page.

HTML elements used:

- Headings
- Paragraphs
- Lists (ordered and unordered)
- Strong (bolding)
- Anchor (links)
- Image
